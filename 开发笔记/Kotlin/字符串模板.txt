原生字符串内，用${}屏蔽字符串同化：
fun main() {
    val vl1:Int=9999;
    println("vl1=$vl1;vl1+1=${vl1+1};12345+54321=${12345+54321}")
    print("我是字符串"+11+(99+1))//原生字符串外，用()屏蔽字符串同化。
}

fun main() {
